# React Native Chat App

1. Clone repository `git clone https://github.com/dloyolar/react-native-chat-app.git`
2. run `yarn install`
3. Set ENV variables (firebase api key):

```
API_KEY=
AUTH_DOMAIN=
PROJECT_ID=
STORAGE_BUCKET=
MESSAGING_SENDER_ID=
APP_ID=
MEASUREMENT_ID=
```

4. Run `yarn start`
